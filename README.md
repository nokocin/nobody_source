# 无名之辈
**NOBODY**

**抗战亲历者记忆库 著**

40位抗战亲历者的最后回忆 一段动荡时代的浮沉身世

他们是士兵，是抵抗者。

之后，他们是解甲者，

是沉默的无名之辈。此刻。

他们是点亮抗战史细部暗处的微光。

9年口述、肖像采集。

不遮蔽、不矫饰。

小角度切入抗战历史纵横面，倾听一个个生命浮沉的微茫声息。
